# http-cli-tool
Simple http tool for Windows 10

## Downloading 
```bash
git clone https://github.com/ramhell220611-ctrl/http-cli-tool.git
cd http-cli-tool
go build -o http-cli.exe cycles.go
```

## Requirements
Go 1.19+

## Supported operations
GET
POST
PUT
DELETE
## You can also work with JSON
